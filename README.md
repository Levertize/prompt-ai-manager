🐤 FlapayBord: A Cute HTML5 Canvas Game

FlapayBord adalah sebuah web-based game yang terinspirasi dari mekanik klasik Flappy Bird, namun dibalut dengan tema karakter chibi yang imut. Game ini dibangun 100% menggunakan Vanilla JavaScript dan HTML5 Canvas tanpa menggunakan game engine pihak ketiga, untuk mendemonstrasikan pemahaman mendalam tentang Game Loop, fisika dasar, dan manipulasi elemen web.

✨ Fitur Utama

Fisika Realistis: Mengimplementasikan kalkulasi gravitasi, momentum jatuh, dan dorongan lompatan yang presisi menggunakan JavaScript.

Deteksi Tabrakan (AABB): Sistem Collision Detection khusus untuk mendeteksi batas karakter dengan rintangan dan lantai.

Kustomisasi Karakter (Skin): Pemain dapat memilih skin karakter favorit mereka di menu utama sebelum permainan dimulai (Demonstrasi manipulasi DOM dan state management).

Game Loop Kustom: Menggunakan requestAnimationFrame untuk memastikan pergerakan frame yang mulus (~60 FPS).

Struktur Proyek Terorganisasi: Pemisahan file yang rapi (/css, /js, /assets) sesuai dengan standar industri pengembangan web.

🕹️ Cara Bermain

Buka link permainan.

Di Menu Utama, klik salah satu dari tiga ikon karakter chibi untuk memilih skin-mu.

Klik tombol "MULAI MAIN!".

Tekan Tombol Spasi di keyboard atau Klik Kiri Mouse untuk membuat karakter melompat.

Hindari rintangan pipa hijau. Jika menyentuh pipa atau jatuh ke lantai, permainan akan berakhir (Game Over).

Klik di mana saja untuk mengulang permainan setelah Game Over.

🛠️ Instalasi Lokal (Bagi Developer)

Jika kamu ingin melihat atau memodifikasi kode game ini di komputermu sendiri:

Clone repositori ini:

git clone [https://github.com/](https://github.com/)Levertize/FlapayBord.git


Buka folder proyek:

cd FlapayBord


Buka file index.html menggunakan browser pilihanmu (Chrome, Firefox, Safari). Tidak perlu menginstal server lokal (Node.js/npm) karena proyek ini menggunakan JavaScript statis murni.
